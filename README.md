# JoystickMounts

The STLs for printing mounts for my joysticks.

I print with white PETg, but PLA will work fine.

Each will require a 1/2"-20 by 3/8" 4 pronged tee nut, such as:

https://amzn.to/3ZbjNyn

POWERTEC QTN1101 4 T 50 Pack | Pronged Tee Nut ¼ Inch-20 & 3/8-Inch – Threaded Inserts for Climbing Holds and Wood Working, 1/4"-20 x 3/8", 50PK
